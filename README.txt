Welcome to offconsole! It's an experemental project made by me, 11 years old Noobisbro(I don't want to tell my real name), for fun. It's 1.0 version, there will be updates.
List of commands:
write -- you can fill one of 3 text values
gen -- generate random number
new -- you are creating a list of 3 cells
fill -- fill cells from first
fillspecial -- fill cell which you have selected
credits -- credits
check -- prints the contents of a variable or a list
calc -- calculator
stop -- stop
cleardisk -- set all to 0
***
All values:
a = text
b = text
c = text
d = number
e = number
f = number
